You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply? -- Example: 3.145.124.47:9000/multiply?num1=3&num2=4
  • /github? -- Example: 3.145.124.47:9000/github?query=users/amehlhase316/repos
  • /randomNumberInRange? -- Example: 3.145.124.47:9000/randomNumberInRange?min=1&max=12
  • /randomDiceRoller -- Example: 3.145.124.47:9000/randomDiceRoller?numberDice=5&dieSize=4
